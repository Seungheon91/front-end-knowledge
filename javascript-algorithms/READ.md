# 시간 복잡도
**시간복잡도**|**설명**|
|:---:|:---:|
|O(1)|상수 형태. n의 값에 상관없이 일정한 양의 계산만 한다.|
|O(logn)|로그 형태|
|O(n)|선형|
|O(nlogn)|선형로그 형태|
|O(n2),O(n3),...|다차 형태|
|O(2n)|지수 형태|
|O(n!)|팩토리얼 형태|
